Comment fonctionne notre Carnet revolutionnaire

Nous avons trois fonctions principal:

La première fonction est de créer un contact dans votre base de données en cliquant sur "Ajouter"
Vous entrez le nom, le prénom, le numero de téléphone et le courriel de la personne que vous désirer ajouter.

Ensuite, nous avons le bouton "Modifier"
Il vous permettera de modifier un contact de votre choix, vous avez juste a cliquer et selectionner le contact dans la liste présenté et modifier les informations.

Vous pouvez aussi voir le bouton "Supprimer"
Ce bouton vous donne la possibilité de supprimer un contact en toute sécurité sans vous casser la tête.

En dernier, le bouton "Actualiser" vous permet de vous assurer que vous avez la dernière version de votre carnet.

Abonnez-vous pour recevoir les mises à jour !
